# Microcontrollers Projects

A collection of **microcontroller and embedded systems projects** using various platforms including **Arduino, STM32, Raspberry Pi**, and other popular development boards.

This repository serves as a central place to organize code, experiments, and projects related to **embedded programming, electronics, and hardware interfacing**.

---

## Platforms

- Arduino
- STM32
- Raspberry Pi
- Other microcontroller boards

---

## Purpose

The goal of this repository is to provide a structured and organized collection of embedded systems projects, helping track progress and share work in one place.
